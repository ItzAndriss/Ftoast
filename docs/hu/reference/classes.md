# Classes referencia

A `classes` egy **string map**, amiben Tailwind classokat tárolsz.

## Toast

| Kulcs | Mit formáz |
|---|---|
| `wrapBase` | wrapper container |
| `toastBase` | a toast kártya |
| `body` | belső padding |
| `row` | fő flex sor |
| `iconBoxBase` | ikon doboz alap |
| `content` | title+message blokk |
| `title` | title tipográfia |
| `message` | message tipográfia |
| `closeBtn` | bezáró gomb |
| `progressTrack` | progress háttér |
| `progressBarBase` | progress sáv |

## Animáció

- `in`: megjelenéskor hozzáadódik
- `out`: eltűnéskor hozzáadódik

## Gombok

- `btnBase`: közös gomb base
- `btn.{variant}`: variánsok

## Dialógus (modal)

| Kulcs | Mit formáz |
|---|---|
| `modalWrap` | fullscreen wrapper |
| `modalBackdrop` | háttér |
| `modalCard` | kártya |
| `modalHeader` | header |
| `modalBody` | body |
| `modalFooter` | footer |
| `modalTitle` | title |
| `modalText` | text |
