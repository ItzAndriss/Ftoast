# FAQ

## Miben más, mint egy klasszik toastr?

- Tailwind alapú (a saját design rendszeredhez igazítható)
- beépített actions gombok
- beépített dialógusok (alert/confirm/prompt)
- i18n + nyelv storage támogatás

## Lehet HTML-t tolni a message-be?

A default render escape-eli a szöveget (biztonság). Ha HTML kell, használj `template()`-et tudatosan.
