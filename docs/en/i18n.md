# i18n + language storage

i18n is used for default titles, aria labels, and dialog button texts.

## Auto language order

1) storage (`storage.key`)
2) `<html lang>`
3) `navigator.language`
4) `fallbackLang`

```js
FToast.config({ storage: { type: "local", key: "site_lang" }, lang: "auto" });
FToast.addTranslations("en", { "dialog.cancel": "Cancel", "close.aria": "Close" });
```
